### Qwen MT翻译插件

翻译软件[pot-app](https://github.com/pot-app/pot-desktop)用于支持千问mt系列翻译模型的插件。

#### 安装方式：

打开`偏好设置`-`服务设置`-`翻译`-`添加外部插件`-`安装外部插件`，选择potext格式插件，然后点击`Qwen MT`设置`API Key`即可。如果需要设置其为默认方式，将其拖拽置顶即可。

#### 功能特性：

- 支持千问MT系列翻译模型（qwen-mt-flash、qwen-mt-turbo、qwen-mt-plus）
- 支持130多种语言翻译
- 可配置模型、API Key、领域提示、temperature参数
- **术语表支持**：可指定自定义术语表JSON文件路径，插件在翻译前自动加载术语并传递给模型，确保专业术语翻译一致性
- **翻译缓存**：可指定SQLite数据库文件路径，插件自动缓存翻译结果，重复翻译时直接返回缓存结果，提高效率
- **强制在线翻译**：可绕过缓存直接请求在线翻译，同时更新缓存数据库

#### 术语表配置：

1. 在插件设置中指定术语表JSON文件路径（如不存在将自动创建空文件）
2. 术语表格式为JSON数组，每个术语包含 `source`（源术语）和 `target`（目标翻译）字段，例如：
```json
[
  {"source": "生物传感器", "target": "biological sensor"},
  {"source": "身体健康状况", "target": "health status of the body"}
]
```

#### 翻译缓存配置：

1. 在插件设置中指定SQLite数据库文件路径（如不存在将自动创建）
2. 插件自动创建 `translations` 表，缓存源文本、源语言、目标语言和翻译结果
3. 相同翻译请求直接返回缓存结果，避免重复调用API
4. 启用“强制在线翻译”可跳过缓存直接请求最新翻译，并更新缓存

#### 打包：

克隆项目后，运行`package.bat`即可。
